# Setup

1. Create a public GitHub repository named exactly `GoofyGoose1`.
2. Upload every file in this package while keeping the folder structure.
3. Replace the LinkedIn and email placeholders in `README.md`.
4. Open **Settings → Actions → General → Workflow permissions** and choose **Read and write permissions**.
5. Open **Actions → Generate contribution snake → Run workflow**.
6. Wait for the `output` branch to appear, then refresh your profile.

The custom header, terminal and footer are local SVG files, so they are not dependent on a third-party banner service.
